    About Bog Language

    Keywords: bogfn, mire, sludge, peat, drain, suckin, fen, gurgle

    Data Types: num, dec, txt, bool

    Print: gurgle("Your message")

    Comments: ~~ single-line, ~~~ ... ~~~ multi-line

    Control Structures:

    • mire (if)
    • sludge (else)
    • peat (for loops)
    • drain (return)

    Example:

    bogfn main(): num {
    fen PI = 3.14
    mire(true) {
        gurgle("Hello Bog World!")
    } sludge {
        gurgle("Nope!")
    }
    drain 0
}
